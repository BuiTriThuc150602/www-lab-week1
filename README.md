# Web Servlet - Các chức năng cơ bản của web site quản lý tài khoản
## các chức năng chính :
### Đăng Nhập
### Đăng ký tài khoản mới
### Quản lý (CRUD) thông tin tài khoản nếu đăng nhập bằng tài khoản quyền admin
## Các công nghệ dùng :
- Java Servlet
- HTML/CSS
- JavaScript
## Database : Sử dụng cơ sở dữ liệu Mariadb
